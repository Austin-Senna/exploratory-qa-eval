# assets

Repository media used by documentation and paper-support workflows.

## Layout

- `images/`: diagrams, logos, and README-facing visual assets.
- `papers/`: local paper PDFs or paper-facing reference files.

Images with a `-white-bg` suffix are white-composited README variants for
Markdown renderers that use dark page backgrounds.

Prefer storing reusable media here instead of leaving image or PDF files in the
repository root. Generated paper figures that are final outputs still belong in
`paper_figures/`.
